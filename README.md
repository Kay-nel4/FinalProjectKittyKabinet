Hello this Final Project titled the Kitty Kabinet is a passion project built on the welfare of community cats.
The goal of this Asp.net Core MVC Web Application is to involve the community in the forever effort to promote Trap Neuter Release (TNR)
through feeding and caring for these cats. The Kabinets will be placed at Walgreens Locations through the area and verified users will have the opportunity to 
donate or take what is needed to feed cat colonies.
This Web App features a Descriptive home page with a clickable link to an image of a prototype of the Kabinet, there is a list of all Kabinets and their contents 
this page has CRUD functionality and can be updated by users. There is also a map with numbered pins that are clickable and lead to specific kabinet details. 
The Kommunity message board keeps everyone updated and also has CRUD functionality.
This entire application is built on a MySql database and includes a CSV file to handle the Locations.
